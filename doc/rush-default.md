NAME
==================================================

**rush default** - Set a default repository.

SYNOPSIS
==================================================

**rush default** REPO

DESCRIPTION
==================================================

Set a default repository.

This command adds (or updates) a repository named **default** to the configuration file, and copies the path from the provided repo.


ARGUMENTS
==================================================

REPO
--------------------------------------------------

Repository name.

- *Required*

EXAMPLES
==================================================

~~~
rush default sample

~~~
