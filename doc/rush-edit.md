NAME
==================================================

**rush edit** - Edit package files.

SYNOPSIS
==================================================

**rush edit** PACKAGE [FILE]

DESCRIPTION
==================================================

Edit package files.

- Alias: **e**

ARGUMENTS
==================================================

PACKAGE
--------------------------------------------------

Package name.

This can either be the package name without the repository name (in this case, the default repository will be used) or in the form of **repo:package**.

- *Required*

FILE
--------------------------------------------------

File to edit.

- Default Value: **main**
